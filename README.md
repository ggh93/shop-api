# 🛒 Shop API

NestJS 기반 커머스 주문/결제 백엔드 API 서버

## 📌 프로젝트 개요

실무 결제 경험을 바탕으로 구현한 커머스 도메인 백엔드 서버입니다.
JWT 인증, 상품/주문 관리, 포트원 결제 연동까지 커머스 핵심 기능을 포함합니다.

## 🛠 기술 스택

- **Framework**: NestJS, TypeScript
- **Database**: MySQL, TypeORM
- **Auth**: JWT, Passport
- **Payment**: 포트원 (iamport)
- **Infra**: Docker, AWS EC2
- **Docs**: Swagger

## ✅ 구현 기능

- [ ] 회원가입 / 로그인 (JWT)
- [ ] 상품 목록 조회 / 상세
- [ ] 주문 생성
- [ ] 포트원 결제 연동
- [ ] 주문 내역 조회
